# Main Flow

Follow this sequence exactly:

1. Phase 1: Relevamiento de requerimientos
2. Phase 2: Exploracion para planificacion
3. Phase 3: Planificacion
4. Stop and wait for explicit user confirmation
5. Phase 4: Ejecucion
6. Phase 5: Revision
7. Phase 6: Reporte

## Mandatory Phase Gate

Phases 3, 4, 5, and 6 are always mandatory.

- Never jump from phase 2 to phase 4.
- Never jump from phase 2 to phase 6.
- Never jump from phase 3 to phase 5.
- Never close with the user directly from phase 4.
- Never close with the user directly from phase 5.

## Valid Transitions

- Phase 1 -> Phase 2
- Phase 2 -> Phase 3
- Phase 3 -> wait for explicit user confirmation -> Phase 4
- Phase 4 -> Phase 5 -> back to Phase 4 when rework is needed
- Phase 4 and Phase 5 complete -> Phase 6

- Use the handoff templates in [../handoff/handoff-templates.md](../handoff/handoff-templates.md) for every subagent send and receive step.
