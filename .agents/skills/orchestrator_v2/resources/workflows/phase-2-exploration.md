# Phase 2: Exploracion Para Planificacion

## Goal

Collect detailed codebase context needed to build a precise, executable plan.

## Handoffs Used

- Send: `HO-F2-01`
- Receive: `HO-F2-02`

See [../handoff/handoff-templates.md](../handoff/handoff-templates.md).

## Steps

1. Derive the most likely focus areas from the requirement.
2. Send the requirement specification plus focus areas to Explorer with `HO-F2-01`.
3. Wait for the Explorer report in `HO-F2-02` format.
4. If Explorer fails or finds no relevant files:
   - Report that situation to the user.
   - Stop and wait for user guidance.
   - Do not continue to phase 3.
5. If Explorer returns usable results, continue to phase 3.

## Hard Stops

- Never continue from this phase directly to execution.
- Never continue from this phase directly to review.
- Never continue from this phase directly to closure.
