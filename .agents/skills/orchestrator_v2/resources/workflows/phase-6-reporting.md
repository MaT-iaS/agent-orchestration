# Phase 6: Reporte

## Goal

Finish the progress file and use it as the final delivery artifact.

## Handoff Used

- `HO-F6-01` in [../handoff/handoff-templates.md](../handoff/handoff-templates.md)

## Steps

1. Gather information from:
   - The execution log in the progress file
   - The plan
   - Coder reports
   - Reviewer reports
2. Complete the report sections above `--- LOG DE EJECUCIÓN ---` in the progress file:
   - Resumen Ejecutivo
   - Informacion del Requerimiento
   - Plan Ejecutado
   - Cambios Realizados
   - Impacto y Side Effects
   - Revision Final
   - Estado Final
   - Recomendaciones, if applicable
3. Update `Fecha ultima actualizacion` in Metadata.
4. Show the finished progress file to the user.
5. Deliver the final summary using `HO-F6-01`.

## Closure Rule

This is the only valid phase for closing the workflow with the user.
