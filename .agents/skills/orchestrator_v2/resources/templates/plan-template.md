# Plan Template

Use this exact plan shape.

```text
Plan Id: PLAN_<NOMBRE>
Descripción: <qué resuelve>
Complejidad estimada: <1-10>
Tareas:
  - <id>: <título>

**Parallel Groups**:
  - Grupo 1: [TASK-01, TASK-03]
  - Grupo 2: [TASK-02]
  - Grupo 3: [TASK-04, TASK-05]
```

**Pasos concretos:**
1. <acción específica 1>

**Riesgos a evitar**: <errores comunes>
